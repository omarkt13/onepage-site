# onepage-site

Product site for OnePage, the e-ink phone companion. Photo-led: the renders
carry the story, one serif line per scene.

- `template.html` — the page source (edit this)
- `assets/` — web-compressed renders (swap/add here)
- `build.py` — inlines assets into a single self-contained `index.html`
- `index.html` — generated output; open in any browser

Rebuild after any change: `python3 build.py`
